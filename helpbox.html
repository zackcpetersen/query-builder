Here is some terminology used in this document.

  • SPREADSHEET is the entire Spreadsheet document, can contain multiple SHEETS and lots of cells / data
  • SHEET is a tab within the SPREADSHEET. There can be many SHEETS in a single SPREADSHEET
  • DRIVE is your Google Drive - where you can store other SPREADSHEETS
  • QUERY is the SQL-like Query where all the magic happens

Steps to use:

  1. Pick the SPREADSHEET you would like to run your Query against - (Use current SPREADSHEET, Select a SPREADSHEET from your Drive)
  2. Insert a range in that SPREADSHEET - Insert the Sheet name, followed by an explanation point (!) and then the cells you wish to include - (Sheet1!A:Z, MySheet!A200:B300, etc..)
  3. Select the columns you would like to be included in the Query - you can use a star (*) to select all of them, or a list of the columns you would like to include separated by commas (A,B,C,D)
  4. Now click “Add New Filter” - here, you’ll need to add one of the columns that you included in the previous step - only use ONE per filter (A).
  5. Then you’ll choose a comparison operator - do you want the value to equal (=), not equal (!=), be greater than (>), less than (<)
  6. Finally, insert the value you would like to compare. Note that =, !=, <, >, <=, >=, can ONLY be used when comparing with numbers, not strings.
  7. Click “ADD” - Congratulations! You’ve made your first filter! Now you can “Add New Filter” again, and they will be used together for more advanced Queries, or click “CREATE QUERY” and you’re done.

Common Issues

Make sure your data types are the same! If one of your filters is A = 1, make sure that in the sheet, 1 is set to either “Automatic” or “Number.”

If your query is for data in another sheet in your drive, you’ll need to swap the column labels (A, B, C, D) with their numbers (Col1, Col2, Col3, Col4)
